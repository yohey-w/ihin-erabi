## カテゴリ1: 物品の整理

> 「60代のうちに少しずつ始めてよかった。70代になってから体力が落ちて、あの時やっておかなかったら大変だったと思う」（X/Twitterより）

物品整理は生前整理の中で最も分量が多い。一気にやろうとせず、部屋・エリアごとに少しずつ進める。

### チェックリスト

- [ ] 衣類: 着ていないものを仕分けする（1年以上着ていないものは処分を検討）
- [ ] 食器・調理器具: 実際に使っているものだけ残す
- [ ] 本・雑誌: 読み返す予定のないものを処分する（図書館への寄付も可）
- [ ] 電化製品: 使っていない・壊れているものを処分する
- [ ] 趣味の道具: 続けているものか確認し、やめたものは処分
- [ ] 寝具・タオル類: 古いもの・予備の多すぎるものを整理
- [ ] 思い出の品: 写真・手紙・日記——残すもの・処分するもの・デジタル化するものを決める

### 判断の基準

「今の自分に必要か」「この先1年で使うか」の2点で判断する。「いつか使うかも」は基本的にNO。

## カテゴリ2: 書類の整理

書類は「何を残すべきか」の判断が難しく、誤って捨てると困るものも多い。

### 残すべき書類チェックリスト

- [ ] 戸籍謄本・住民票（最新版は常に取り直せるが、古いものは確認）
- [ ] マイナンバーカード・運転免許証・パスポート
- [ ] 保険証書（生命保険・医療保険・火災保険）
- [ ] 不動産の登記済権利証・登記識別情報
- [ ] 預金通帳・ローン契約書
- [ ] 年金手帳・年金証書
- [ ] 遺言書（自筆証書遺言の場合は保管先を家族に伝える）
- [ ] エンディングノート

### 書類の整理方法

書類は「重要書類フォルダ」にまとめて、家族が分かる場所に保管する。「大事な書類はここにある」と家族に伝えておくことが重要だ。

## カテゴリ3: デジタルの整理

スマホ・PC・インターネットサービスのデジタル遺品は、近年特に重要度が増している。

### チェックリスト

- [ ] スマホのパスコードを家族に伝えるか、安全な場所に記録する
- [ ] PCのパスワードを記録する
- [ ] 重要なアカウント（メール・SNS・ネット銀行）のID・パスワードを一覧化する
- [ ] サブスクリプションの一覧と解約方法を記録する
- [ ] クラウドストレージ（写真・書類）の保存場所と共有設定を確認する
- [ ] 不要なアカウントは自分で解約しておく

### パスワード管理の方法

紙のノートに記録する方法と、パスワード管理アプリを使う方法がある。紙の場合は保管場所に注意（他人に見られない場所）。アプリを使う場合はマスターパスワードを家族に伝えておく。

## カテゴリ4: 金融・資産の整理

相続手続きで最も時間がかかるのが金融資産だ。事前に整理しておくと遺族の負担が大きく減る。

### チェックリスト

- [ ] 銀行口座の一覧（金融機関名・支店名・口座番号）を作成する
- [ ] 証券口座・投資信託の一覧を作成する
- [ ] 保険の一覧（保険会社・証書番号・受取人）を作成する
- [ ] 不動産の情報（所在地・地番・固定資産税の情報）を整理する
- [ ] ローン・借金の情報（残高・返済先）を整理する
- [ ] 財産目録を作成し、安全な場所に保管する

### 遺言書の検討

財産の分け方に希望がある場合は、遺言書の作成を検討する。公正証書遺言は公証役場で作成でき、紛失・改ざんの心配がない。

 > 「父が亡くなったとき、エンディングノートがあった。銀行の一覧も保険の一覧も書いてあって、あれがなかったら手続きに何倍も時間がかかっていたと思う」（X/Twitterより）

## カテゴリ5: 人間関係の整理

「連絡してほしい人・してほしくない人」を整理しておくと、残された家族の負担が減る。

### チェックリスト

- [ ] 連絡先一覧（住所録）を更新する
- [ ] 「死亡時に連絡してほしい人のリスト」を作成する
- [ ] 「知らせなくていい人・近況を伝える必要のない人」を整理する
- [ ] SNSアカウントの対応方法（削除か追悼か）を決めておく
- [ ] ペットがいる場合: 自分が先に亡くなった場合の預け先を決めておく

## カテゴリ6: 医療・介護情報の整理

緊急時・介護が必要になった時のために準備しておく情報だ。

### チェックリスト

- [ ] かかりつけ医・通院中の病院の情報を記録する
- [ ] 服用中の薬の情報（薬の名前・用量・処方元）を記録する
- [ ] 血液型・アレルギー・持病の情報を記録する
- [ ] 救急・緊急時に連絡してほしい人を指定する
- [ ] 延命治療に関する希望（意思決定文書・リビングウィル）を作成・保管する
- [ ] 介護が必要になった場合の希望（施設か在宅か）を家族に伝えておく

> 「エンディングノートを書いていて、自分の財産が思ったよりも複雑に分散していることに気づいた。整理しておいてよかった」（X/Twitterより）

## 生前整理 優先度チェックリスト

何から始めるか迷ったときに使えるシンプルな優先度表だ。

| カテゴリ | 緊急度 | 理由 |
|---|---|---|
| 書類（通帳・保険証書・権利証） | 高 | 紛失すると相続手続きが困難になる |
| 金融・資産の一覧化 | 高 | 未発見のまま時効になるリスクがある |
| デジタル遺品（ID・パスワード） | 高 | 知らないと家族がアクセスできなくなる |
| 物品の整理 | 中 | 量が多いほど後で家族の負担になる |
| 人間関係（連絡先・SNS） | 中 | 残された家族が対応に困るケースがある |
| 医療・介護情報 | 中 | 緊急時に役立つ。元気なうちに記録を |

## ケース別——生前整理チェックの始め方

状況によって優先すべきことが変わる。あなたに近いケースを確認してほしい。

### ケース1: 本人が元気で自分で進められる

体力と判断力がある今こそ、生前整理を始める最適なタイミングだ。生前整理を始める最適な年齢は60代前半と言われており、元気なうちにスタートすることで費用もかからず自分のペースで進められる。

| 優先順 | やるべきこと | ポイント |
|--------|------------|---------|
| 1 | エンディングノートの記入から開始 | 財産・医療情報の整理と意思表示が一度にできる |
| 2 | 1日30分ずつ不要品を仕分け | 週末だけでも3ヶ月で大きく前進できる |
| 3 | デジタル遺品リストの作成 | アカウント・パスワード一覧を安全な場所に保管 |

### ケース2: 本人の判断力が低下している

家族が主導で進める場合は、法的な効力を持つ書類の作成を最優先にする。

| 優先順 | やるべきこと | ポイント |
|--------|------------|---------|
| 1 | 家族が主導で最低限の書類整理 | 通帳・保険証書の場所だけでも把握する |
| 2 | 法的書類（遺言・委任状）を早めに作成 | 判断能力があるうちでないと作成できない |
| 3 | 専門業者への相談を検討 | 一人で抱え込まず、サポートを借りる選択肢も |

## 生前整理の進め方——焦らない・一人でやらない

### 全部を一気に終わらせようとしない

生前整理は長期的な取り組みだ。週に一度、30分から1時間程度を継続する方が無理なく進められる。

### 家族と一緒に進める

一人でこっそり進めるより、家族に「今生前整理をやっている」と伝えておく方が良い。共有することで、「どこに何があるか」の情報伝達にもなる。

### 専門家・業者を活用する

荷物の処分に困ったら業者に、書類の整理・相続の準備に困ったら司法書士・行政書士に相談する選択肢がある。
